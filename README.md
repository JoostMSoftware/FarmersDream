# Farmer's Dream

An addon for Farmer's Delight to enhance your farming needs.

## Features

### Current features

* new crops & seeds
* new food recipes

### Planned Features

* new crops, seeds and food recipes
* Fermenting station
* A way to make Farmer's Delight crops and Farmer's Dream crops work with the fermenting station
* Bio fuel generator
